Pour créer un dépot git:   
git init dans le dossier, puis git remote add origin https://github.com/mmarie4/PROJET_PRS (ensuite on peut push)   
Pour cloner un dépot:   
git clone https://github.com/mmarie4/PROJET_PRS   
A chaque modif:   
git add *   
git commit -a   
git pull   
git push origin master
